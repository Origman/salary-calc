## Что это за проект

Это одностраничный калькулятор зарплаты и НДФЛ для России на 2026 год.

Сайт умеет:
- считать `gross -> net`;
- считать `net -> gross`;
- показывать эффективную ставку НДФЛ;
- показывать годовое распределение дохода по ставкам НДФЛ;
- показывать помесячный расчет: ставка(и), НДФЛ и сумму на руки;
- переключать валюту отображения: `RUB / USD / EUR`.

Целевая модель расчета:
- резидент РФ;
- доходы от трудовой деятельности / ГПХ;
- без налоговых вычетов, льгот и специальных режимов.

## Что уже реализовано

- Прогрессивная шкала НДФЛ:
- `13%` до `2 400 000` руб. в год
- `15%` от `2 400 000` до `5 000 000`
- `18%` от `5 000 000` до `20 000 000`
- `20%` от `20 000 000` до `50 000 000`
- `22%` свыше `50 000 000`

- Два режима ввода:
- `Гросс (до НДФЛ)`
- `Нет (на руки)`

- Два периода:
- `в месяц`
- `в год`

- Валюты:
- `RUB`
- `USD`
- `EUR`

- Курс валют:
- загружается с официального источника ЦБ РФ через серверный endpoint `/api/rates`;
- в интерфейсе показывается дата курса и значения `USD` / `EUR`;
- все внутренние расчеты ведутся в рублях, валюта влияет на ввод и отображение.

- Сохранение состояния через cookie:
- сумма в поле ввода;
- режим `gross / net`;
- период `month / year`;
- валюта `RUB / USD / EUR`.

- Поведение поля ввода:
- автоматически добавляются разделители тысяч;
- при смене валюты значение в поле тоже конвертируется;
- это исправляет прошлую проблему, когда менялся только знак валюты, а число оставалось тем же.

- В таблице годового распределения скрываются строки `20%` и `22%`, если по ним нули.

## Архитектура

Основные файлы:
- `index.html` — весь UI и клиентская логика калькулятора.
- `api/rates.js` — serverless endpoint Vercel, который берет XML курсов с сайта ЦБ РФ и отдает JSON.
- `favicon.svg` — иконка сайта для вкладки браузера.
- `.vercel/project.json` — привязка локальной папки к Vercel-проекту.
- `backup.html` — старый снимок страницы.

Ключевые части в `index.html`:
- `const BREAKS` — налоговые пороги и ставки.
- `annualGrossFromAnnualNet()` — обратный расчет `net -> gross`.
- `annualBreakdown()` — распределение дохода по ставкам.
- `monthlySchedule()` — помесячный расчет по накопленной базе.
- `fetchCurrentRates()` — загрузка свежих курсов через `/api/rates`.
- `persistSalaryInput()` / `persistSelection()` / `restoreSelection()` — cookies.
- `formatEditableInput()` / `formatSalaryInputField()` — форматирование поля ввода.
- `syncInputToCurrency()` — конвертация значения поля при смене валюты.

Ключевые части в `api/rates.js`:
- запрос к `https://www.cbr.ru/scripts/XML_daily.asp`;
- парсинг `USD` и `EUR`;
- возврат JSON вида:

```json
{
  "source": "CBR",
  "date": "18.03.2026",
  "rates": {
    "USD": 81.9103,
    "EUR": 93.1557
  }
}
```
