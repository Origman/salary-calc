# salary-calc

Одностраничный калькулятор зарплаты и НДФЛ для России на 2026 год.

Продакшн: [sal-calc.vercel.app](https://sal-calc.vercel.app)

## Что умеет

- считать `gross -> net`
- считать `net -> gross`
- показывать эффективную ставку НДФЛ
- показывать годовое распределение дохода по ставкам
- строить помесячный расчет: гросс, НДФЛ, сумма на руки и эффективная ставка
- переключать отображение в `RUB / USD / EUR`
- загружать актуальные курсы `USD / EUR` через `CBR`
- сохранять состояние формы в cookie

## Для кого рассчитан

Калькулятор ориентирован на:

- налогового резидента РФ
- доходы от трудовой деятельности или ГПХ
- расчет без вычетов, льгот и специальных режимов

## Налоговая модель

Используется прогрессивная шкала НДФЛ:

- `13%` до `2 400 000` ₽ в год
- `15%` с `2 400 000` до `5 000 000` ₽
- `18%` с `5 000 000` до `20 000 000` ₽
- `20%` с `20 000 000` до `50 000 000` ₽
- `22%` свыше `50 000 000` ₽

## Как устроен проект

Основные файлы:

- `index.html` — интерфейс и вся клиентская логика калькулятора
- `api/rates.js` — serverless endpoint Vercel для курсов ЦБ РФ
- `favicon.svg` — иконка сайта
- `PROJECT_CONTEXT.md` — рабочий контекст проекта

Ключевые части клиентской логики:

- `BREAKS` — пороги и ставки НДФЛ
- `annualGrossFromAnnualNet()` — обратный расчет `net -> gross`
- `annualBreakdown()` — годовое распределение по ставкам
- `monthlySchedule()` — помесячный расчет по накопленной базе
- `syncInputToCurrency()` — поведение суммы при смене валюты
- `persistSelection()` и `restoreSelection()` — сохранение пользовательских настроек

## Локальный запуск

Для простого просмотра можно открыть файл напрямую:

```powershell
start "" "D:\AI projects\salary calc\index.html"
```

Но для корректной работы загрузки курсов лучше запускать через локальный HTTP-сервер или Vercel dev.

Пример с Vercel:

```powershell
npx vercel dev
```

После этого откройте локальный адрес, который покажет Vercel.

## Деплой

Продакшн-деплой:

```powershell
npx vercel --prod --yes
```

Проверка endpoint курсов:

```powershell
Invoke-WebRequest https://sal-calc.vercel.app/api/rates
```

## Источник курсов

Курсы валют берутся с сайта ЦБ РФ:

- `https://www.cbr.ru/scripts/XML_daily.asp`

Endpoint `/api/rates` возвращает JSON в формате:

```json
{
  "source": "CBR",
  "date": "18.03.2026",
  "rates": {
    "USD": 81.9103,
    "EUR": 93.1557
  }
}
```

## Ограничения

Сейчас в проекте нет:

- учета налоговых вычетов
- режимов для нерезидентов
- сохранения нескольких сценариев расчета
- экспорта результатов
- ручного выбора даты курса
